Along with the Fluxnova web applications we ship Admin, accessible via [http://localhost:8080/fluxnova/app/admin/](http://localhost:8080/fluxnova/app/admin/).
Admin is an application that allows you to configure users and groups via the engine's [Identity Service]({{< ref "/user-guide/process-engine/identity-service.md" >}}) and authorizations via the engine's [Authorization Service]({{< ref "/user-guide/process-engine/authorization-service.md" >}}). Furthermore, you can connect Fluxnova Admin to your [LDAP system]({{< ref "/user-guide/process-engine/identity-service.md#the-ldap-identity-service" >}}).
